# Vexura - Salary & Income Tax Calculator

A comprehensive web application for calculating salary taxes and income taxes on an annual basis. Built with Nuxt 3, Vexura helps customers understand their tax obligations and net income based on their salary information.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (recommended) or npm

### Installation

Install the dependencies:

```bash
pnpm install
```

### Development

Start the development server on `http://localhost:3000`:

```bash
pnpm run dev
```

### Production

Build the application for production:

```bash
pnpm run build
```

Preview the production build:

```bash
pnpm run preview
```

## Usage

1. Navigate to the application in your browser
2. Enter your annual salary information
3. View detailed tax calculations and breakdowns
4. Understand your net income after taxes
